Introduction
============

Welcome


Quick start
-----------

0. Bootstrap

.. code:: bash

   $ python bootstrap.py
   $ bin/buildout

1. Start server

.. code:: bash

   $ make server

3. Build docs

.. code:: bash

   $ make robot

Standalone build
----------------

.. code:: bash

   $ make html

Resetting generated screenshots
-------------------------------

.. code:: bash

   $ make clean
